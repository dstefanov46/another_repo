PV Forecaster Documentation
===================================

PV Forecaster is a Python app for prediction of PV generation across Slovenia. It was developed for the Slovenian
Transmission Network Operator ELES.

The section :doc:`installation` includes information on how to install the **conda** environment in which the
*PV Forecaster* was developed, and in the section :doc:`code` you can find description of the app code.

.. note::

   This project is under active development.
